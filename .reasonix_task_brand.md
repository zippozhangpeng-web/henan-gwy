# 好风公考 — 品牌主页改造

## 目标
将现有的首页（数据仪表盘）改造成好风公考品牌主页，突出培训机构特色，同时保留情报站的明显入口。

## 现首页模板
`templates/index.html` — 当前是和搜索、推荐、统计图表结合的数据面板。改造后：
- 品牌宣传内容在前（好风机构介绍）
- 情报站在后（保留推荐岗位+大数据卡片+快速检索入口）
- 新增学员意向登记模块

## 设计原则
- 保持现有橙色调渐变风格（`--primary: #FF6B35`, `--accent: #FF3366`）
- 保持现有的卡片/按钮/动画风格
- 不修改 base.html（导航栏、页脚不变）
- 不修改 CSS 变量，仅新增需要的样式

- 推荐岗位（综合评分TOP6）→ 如果时效性允许
- 快速搜索入口（简化版，只有年份+专业+学历三字段）

**位置：** 页面底部，情报站模块之后

**设计：**
```

│ 📝 免费获取选岗方案                            │
│                                               │

│  电话      [___________]                      │
│  意向地市   [___________] (下拉选择18个地市)    │
│  意向岗位   [___________] (文本，如：公安/乡镇) │
│                                               │
│  [提交咨询 🚀]                                 │
│                                               │
│  💬 或直接联系我们：微信 XXX-XXXX-XXXX          │
└──────────────────────────────────────────────┘
```

**后端要求：**
1. 新增 SQLite 表 `leads`（id, name, phone, city, position_type, created_at, status）
2. 新增 POST 路由 `/api/lead` 接收表单提交
3. 提交成功后返回 JSON `{"status": "ok", "message": "已收到您的咨询，我们将在24小时内与您联系"}`
4. 前端 AJAX 提交，成功后显示成功提示
