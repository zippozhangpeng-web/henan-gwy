# 好风公考情报站 - 前端改造任务

## 项目概述
Flask + Jinja2 + Bootstrap 5 网站，数据库已有31,541条真实岗位数据+7,295条进面分数+31,541条四维评分。

## 数据API
- `app.py` 中 `/` 路由返回 `stats` 对象，包含:
  - `stats.total_positions` / `stats.total_recruit`
  - `stats.by_year` / `stats.by_city` / `stats.by_system` / `stats.by_edu`
  - `stats.hardest` / `stats.easiest` - 按招录人数排序的岗位TOP5(有id/unit/city/recruit_num字段)
- `position_scores` 表: `position_id, difficulty_score, region_score, salary_score, prospect_score, overall_score`
- 可通过 `/position/<id>` 路由查看单岗位详情
- `utils/query.py` 中 `get_stats()` 已有所有统计数据

## 技术栈
- Flask + Jinja2
- Bootstrap 5 (CDN: cdn.jsdelivr.net)
- ECharts 5 (CDN: cdn.jsdelivr.net)
- CSS: `/static/css/style.css` (现有样式变量: --primary, --secondary, --accent等)
- JS: `/static/js/charts.js` (已有 initCitiesBarChart 等函数)
